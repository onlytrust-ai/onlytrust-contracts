# Security Policy

## Smart Contract Vulnerabilities

**Do not report smart contract vulnerabilities through public GitHub issues.**

Email: **security@onlytrust.ai**

Include:
- Contract and function affected
- Description of the vulnerability
- Steps to reproduce / proof of concept
- Potential impact (fund loss, DoS, etc.)
- Suggested fix (if any)

## Response Timeline

| Stage | Timeframe |
|-------|-----------|
| Acknowledgment | 48 hours |
| Assessment | 5 business days |
| Fix & Disclosure | 30 days (coordinated) |

## Audit Status

Contracts are currently **unaudited** and deployed only on Base Sepolia testnet.

## Bug Bounty

No formal bug bounty program at this time. We appreciate responsible disclosure.
